# 大学城美食地图 - UI规范文档

---

## 一、设计概述

### 1.1 设计理念

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   "为大学生打造的美食探索工具，既有年轻活力，又不失实用优雅"                    │
│                                                                             │
│   核心设计关键词:                                                            │
│   ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐                  │
│   │  清透     │  │  活力     │  │  亲和     │  │  高效     │                  │
│   │  Clarity │  │  Energy  │  │  Friendly│  │  Efficient│                  │
│   └──────────┘  └──────────┘  └──────────┘  └──────────┘                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 风格定位：粘土轻拟物 + 青年活力

- **粘土质感**：柔和的阴影、圆润的边角、轻微的立体感
- **轻拟物**：不追求完全写实，点到为止的质感暗示
- **青年活力**：明快的配色、流畅的动效、有趣的微交互
- **清透感**：充足的留白、清晰的层次、不拥挤的布局

### 2.2 页面结构说明

| 区域 | 位置 | 尺寸 | 功能说明 |
|-----|------|------|---------|
| **顶部导航栏** | 顶部固定 | 64px高，100%宽 | Logo、城市切换、搜索框、消息、用户头像 |
| **左侧导航栏** | 左侧固定 | 200px宽 | 附近美食、热门推荐（含子标签）、我的收藏、场景标签 |
| **主地图区** | 中央 | 自适应 | 全屏交互地图，商家标记带拥挤度红环 |
| **底部Sheet** | 底部可拖拽 | 默认收起，可展开 | 快捷筛选、商户卡片流、排行榜/最近浏览Tab |
| **右侧详情页** | 右侧滑出 | 50%屏幕宽度 | 商家详情、实时状态、路线规划、客流趋势 |
| **Footer** | 底部 | 小字 | 版权信息、关于我们、隐私政策 |

### 2.3 交互逻辑

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                           主页交互逻辑                                        │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  1. 地图标记点击                                                              │
│     点击地图上的商户标记 → 右侧滑出50%宽度详情页                              │
│     标记显示拥挤度红环：绿色(空闲) 黄色(忙碌) 红色(满客) 紫色(排队)           │
│                                                                              │
│  2. 底部Sheet交互                                                             │
│     默认收起状态 → 向上拖拽展开 → 显示完整商户列表                            │
│     展开后：                                                                  │
│     • 顶部：快捷筛选栏（价格/距离/类型）                                      │
│     • 中部：横向滑动商户卡片流                                                │
│     • 底部：Tab切换（排行榜 / 最近浏览）                                      │
│                                                                              │
│  3. 左侧导航点击                                                              │
│     • 附近美食 → 地图定位到当前位置附近                                       │
│     • 热门推荐 → 展开子标签 [今日热榜] [本周排行]                             │
│     • 我的收藏 → 地图显示收藏的商家                                           │
│     • 场景标签 → 按场景筛选商家（课间快吃/宿舍聚餐等）                        │
│                                                                              │
│  4. 搜索交互                                                                  │
│     点击搜索框 → 展开搜索建议面板 → 输入关键词 → 显示搜索结果                 │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 2.4 响应式断点

| 断点 | 宽度范围 | 布局调整 |
|-----|---------|---------|
| **Desktop XL** | ≥1440px | 左侧导航200px + 全屏地图 + 右侧详情50% |
| **Desktop** | 1280px-1439px | 左侧导航200px + 全屏地图 + 右侧详情50% |
| **Tablet** | 768px-1279px | 左侧导航收起为图标栏(64px)，底部Sheet全宽 |
| **Mobile** | <768px | 底部导航栏，左侧变为抽屉，地图全屏 |

## 六、微交互与动效

### 6.1 动效原则

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           动效设计原则                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1. 有目的的动效                                                            │
│     • 引导注意力 → 新内容加载、状态变化                                      │
│     • 提供反馈 → 点击、悬停、操作结果                                        │
│     • 增强理解 → 空间关系、层级结构                                          │
│                                                                             │
│  2. 轻量快速                                                                │
│     • 时长: 150-300ms (微交互) / 300-500ms (页面过渡)                        │
│     • 缓动: ease-out (进入) / ease-in-out (对称)                            │
│                                                                             │
│  3. 一致性                                                                  │
│     • 相同组件使用相同动效                                                   │
│     • 动效方向符合物理直觉                                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 6.2 微交互清单

### 9.1 图标库建议

- **主要**: Phosphor Icons (https://phosphoricons.com/)
- **备选**: Heroicons (https://heroicons.com/)
- **风格**: Regular + Fill 双版本，24px标准尺寸

### 9.2 图片规范

| 类型 | 尺寸 | 格式 | 大小限制 |
|-----|------|-----|---------|
| 商家封面 | 800x500 | WebP/JPG | <200KB |
| 商家Logo | 200x200 | PNG/WebP | <50KB |
| 菜品图片 | 600x400 | WebP/JPG | <150KB |
| 用户头像 | 200x200 | JPG | <50KB |
| 评价图片 | 800x800 | WebP/JPG | <200KB |
