# 🍜 大学城美食地图

> 校友吃什么，你看就知道

一个为大学生打造的本地化美食探索工具，帮助用户快速发现大学城周边餐饮商户，查看实时客流，做出就餐决策。

## ✨ 功能特性

### C端用户侧
- 🗺️ **地图探索** - 基于高德地图的周边商家可视化展示
- 📍 **实时状态** - 查看商家当前拥挤程度（空闲/忙碌/满客/排队）
- 🔥 **热门推荐** - 今日热榜、本周排行
- ❤️ **收藏管理** - 收藏喜欢的商家
- 🔍 **智能搜索** - 按类型、场景、关键词搜索
- ⭐ **评价系统** - 校友真实评价，可信度高
- 👤 **个人中心** - 管理个人信息、收藏、评价

### B端商家侧
- 📊 **数据看板** - 访客统计、收藏数据、评分监控
- 📍 **实时状态** - 快速更新店铺状态、排队人数
- 🍽️ **菜品管理** - 管理店铺菜品信息（开发中）
- 💬 **评价管理** - 查看和回复用户评价（开发中）

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 初始化数据库
```bash
npm run init-db
```

### 启动服务器
```bash
npm start
```

服务器将在 http://localhost:3000 启动

## 🗄️ 数据库设计

### 核心表
- `universities` - 大学信息
- `merchants` - 商家信息
- `products` - 商品/菜品
- `users` - 用户信息
- `reviews` - 评价信息
- `user_favorites` - 用户收藏
- `regions` - 商圈/区域

## 📝 数据说明

项目使用北京市学院路大学城作为数据基础：
- 覆盖5所高校：清华、北大、北航、北邮、中农
- 5个商圈区域
- 25家真实类型商家数据
- 包含快餐、中餐、饮品、小吃等多种类型

## ⚠️ 注意事项

1. **高德地图API**: 项目需要配置高德地图Key才能正常使用地图功能。请在 `client/pages/home.html` 中替换 `您的高德地图Key`。

2. **浏览器访问**: 项目设计为纯Web端，建议使用桌面浏览器访问，推荐分辨率1280x720以上。

3. **数据库**: 项目使用SQLite作为数据库，数据文件存储在 `data/campus_food.db`。
