## 一、产品概述

### 1.1 产品定位
为大学生打造的本地化美食探索工具，帮助用户快速发现大学城周边餐饮商户，查看实时客流，做出就餐决策。

**Slogan**: "校友吃什么，你看就知道"

### 1.2 目标用户
- 核心用户：18-25岁在校大学生
- 使用场景：日常快餐、宿舍聚餐、约会用餐、深夜食堂

### 1.3 核心价值
1. **发现美食** - 地图可视化展示周边商户
2. **快速决策** - 实时状态、评分、距离一目了然
3. **真实口碑** - 校友评价，可信度高

## 二、数据库设计（轻量版）

### 2.1 核心表清单

| 模块 | 表名 | 用途 |
|-----|------|------|
| 用户 | users | 用户基础信息 |
| 用户 | user_preferences | 用户偏好设置 |
| 用户 | user_actions | 用户行为日志 |
| 用户 | user_favorites | 用户收藏 |
| 商家 | merchants | 商家完整信息 |
| 商家 | merchant_status_logs | 状态变更日志 |
| 商家 | merchant_business_hours_special | 特殊营业时间 |
| 商品 | products | 商品/菜品 |
| 商品 | product_categories | 商品分类 |
| 地区 | regions | 地理区域 |
| 地区 | region_heat_stats | 热力统计 |
| 地区 | region_reviews | 地区评价 |
| 评价 | reviews | 用户评价 |
| 学校 | universities | 大学信息 |
| 学校 | campuses | 校区信息 |

### 2.2 核心字段说明

**merchants（商家表）关键字段：**
- location: PostGIS地理点，存储经纬度
- realtime_status: 实时状态（1空闲/2忙碌/3满客/4排队/5休息中）
- queue_count: 当前排队人数
- scene_tags: 场景标签JSON（["约会","聚餐","快餐"]）
- review_avg_score: 平均评分
- price_per_person: 人均消费

**region_heat_stats（热力统计表）关键字段：**
- stat_date/stat_hour: 统计日期和小时
- customer_count: 客流量
- heat_level: 热力等级（1冷/2温/3热/4爆）

## 五、API接口需求

### 5.1 首页接口

**获取附近商家**
```
GET /api/merchants/nearby
参数:
  - lat: 纬度
  - lng: 经度
  - distance: 距离范围(米)
  - type: 商家类型
  - price_min/max: 价格范围
返回:
  - 商家列表（含位置、评分、状态）
```

**获取热力数据**
```
GET /api/regions/heatmap
参数:
  - region_id: 区域ID
  - date: 日期
返回:
  - 热力统计数组
```

### 5.2 商家接口

**获取商家详情**
```
GET /api/merchants/:id
返回:
  - 商家完整信息
  - 实时状态
  - 评分统计
```

**获取商家菜品**
```
GET /api/merchants/:id/products
返回:
  - 菜品列表
```

**获取商家评价**
```
GET /api/merchants/:id/reviews
参数:
  - page: 页码
  - type: 评价类型(good/mid/bad/with_image)
返回:
  - 评价列表
```

### 5.3 用户接口

**用户登录**
```
POST /api/auth/login
参数:
  - code: 微信授权码
返回:
  - token
  - 用户信息
```

**收藏/取消收藏**
```
POST /api/favorites
参数:
  - merchant_id: 商家ID
  - action: add/remove
```

**提交评价**
```
POST /api/reviews
参数:
  - merchant_id: 商家ID
  - rating: 评分
  - content: 内容
  - images: 图片数组
```

## 六、数据需求

### 6.1 种子数据范围
- 地区：北京学院路周边
- 大学：5所（清华、北大、北航、北邮、中农）
- 商圈：4-5个（五道口购物中心、华联商厦等）
- 商家：约100家

### 6.2 数据类型分布
| 类型 | 数量 | 示例 |
|------|------|------|
| 快餐 | 25 | 麦当劳、肯德基、吉野家 |
| 中餐 | 30 | 川菜、粤菜、东北菜 |
| 饮品 | 15 | 星巴克、喜茶、瑞幸 |
| 小吃 | 20 | 炸酱面、煎饼、烤串 |
| 其他 | 10 | 火锅、烧烤、日料 |
